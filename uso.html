Cómo usar la API de Productos

Esta guía explica cómo usar la API de productos sin entrar en detalles técnicos.

1. ¿Qué puedes usar?

La API te permite acceder al listado de productos y ver cada producto individual. También ofrece respuestas simuladas cuando se usan otros métodos HTTP como POST, PUT o DELETE.

2. Endpoints disponibles

  • GET https://d3m4rc0.github.io/API-Productos-Resto/productos.json
  • GET https://d3m4rc0.github.io/API-Productos-Resto/api/productos
  • GET https://d3m4rc0.github.io/API-Productos-Resto/api/productos/1 (para un producto específico)

3. Métodos HTTP adicionales

Si se usa POST, PUT o DELETE, la API devolverá una respuesta automática. Estas peticiones son útiles para probar el comportamiento del frontend, aunque no modifican datos reales.

  • POST a /api/productos
  • PUT a /api/productos/1
  • DELETE a /api/productos/1

4. ¿Qué verás en la respuesta?

Las respuestas para métodos distintos a GET te dirán que la petición fue recibida y mostrarán el tipo de operación. Eso permite hacer pruebas sin necesidad de un servidor real.

5. Recomendación

Abre primero la página principal del sitio para activar la API simulada. Luego podrás usar cualquiera de los endpoints desde tu frontend.
