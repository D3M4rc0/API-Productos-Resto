¿Qué se muestra aquí?

Esta página es un index de ayuda para tu API estática. Si prefieres, puedes eliminarla y la raíz mostrará directamente el JSON o el README según la configuración de Pages.

Importante: al mantener este index.html, la raíz de tu sitio / muestra esta página en lugar del README.

Si quieres que la raíz sea solo el JSON, puedes eliminar este archivo y volver a publicar Pages.
