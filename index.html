API Simulada de Productos para e-Commerce Resto Bar

Este repositorio expone los datos desde productos.json y las imágenes desde img/productos/.

Endpoint JSON

Usa el siguiente URL para consumir los datos desde el frontend: 

https://raw.githubusercontent.com/D3M4rc0/API-Productos-Resto/main/productos.json

Si GitHub Pages está publicado correctamente, el JSON también debería ser accesible en: 

https://d3m4rc0.github.io/API-Productos-Resto/productos.json

Ejemplo de consumo

fetch('https://raw.githubusercontent.com/D3M4rc0/API-Productos-Resto/main/productos.json')
  .then(res => res.json())
  .then(productos => {
    console.log(productos);
  });

Ejemplo de imagen

Usa la ruta directa a la carpeta de imágenes: 

https://raw.githubusercontent.com/D3M4rc0/API-Productos-Resto/main/img/productos/arepas-carne.webp

HTML ejemplo:

<img src="https://raw.githubusercontent.com/D3M4rc0/API-Productos-Resto/main/img/productos/arepas-carne.webp" alt="Arepas con Carne" />

Notas

  • Este sitio estático es solo lectura (GET).
  • Los productos se actualizan directamente en productos.json.
  • No hay backend ni endpoints dinámicos adicionales.
